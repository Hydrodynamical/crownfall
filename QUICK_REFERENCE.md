# CROWNFALL — One-Page Reference

## Turn Order
1. **DRAW 1** (mandatory). Deck empty → **BLEED**: 1 wound first time, **2 wounds** each time after.
2. **UP TO 2 ACTIONS**, any mix — each stack attacks max once per your turn; blocking is always free:

| Action | Effect |
|---|---|
| **MUSTER** | Agent from hand → Front, face down (max 3 stacks) |
| **ARM** | Augment from hand → under a stack, face down (respect slots) |
| **STRIKE** | Attack the enemy **player** with a stack |
| **CHALLENGE** | Attack an enemy **stack** with a stack |
| **RECALL** | Return one of your stacks to hand |

*Game turn 1 only: first player takes 1 action, no attacks.*

## Ranks
| Rank | Value | Slots | Ceiling |
|---|---|---|---|
| A | 14 | 0 | 14 — survives ties |
| K | 13 | 1 | 25 — Decree: flip up to forbid retreat |
| Q | 12 | 2 | 35 |
| J | 11 | 3 | 44 |

**Stack size is public:** 4 cards = J · 3 = Q/J · 2 = K/Q/J · 1 = anything.

## Combat
- **Strike** → defender **blocks** (battle) or **takes wounds = striker's card count** (no reveal; striker stays).
- **Challenge** → defender **fights** or **retreats** (stack to hand unrevealed; pay 1 of its augments to own Wounds; lone agent free). **Decree King** = no retreat.
- **Raise** (battle locked, before reveal): defender first, each may add ≤1 augment from hand, face down, slots permitting.
- **Reveal & total:** `agent + augment pips + 2 per suit-matched augment (Loyalty)`.
- **Higher total wins.** Loser's whole stack → loser's Wounds (agent dead forever). Winner spends **exactly 1 augment** → own Wounds; rest returns to hand; suit power triggers. All one simultaneous step; **death checked after**.
- **Tie:** both stacks destroyed (Aces return to hand instead).
- **Illegal stack revealed** (no agent on top / over slots): auto-loses the battle.

## Wound Source Order (recheck each wound)
**1.** top of deck → **2.** hand, opponent pulls blind → **3.** Front, augments before any agent.

## Suit Powers (on the agent leading the stack)
| | Court | Power |
|---|---|---|
| ♠ | Legion | Battle win → loser takes **+1 wound** |
| ♥ | Cloister | Battle win where you spent an augment → return a **different** number card from your Wounds to hand |
| ♦ | Exchange | Battle win → **spend nothing**; augments return (but were seen) |
| ♣ | Veil | First muster of each Club agent → **peek** one face-down card in an enemy Front stack (named by position) |

**Loyalty (all suits):** each augment matching its agent's suit counts **+2**.

## Death
You lose at **13 wounds**, or with **no agents** in hand + Front. Both defeated after a battle: fewer wounds wins; tied → **active player wins**.
